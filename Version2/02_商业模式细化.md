## 一、商业模式画布

### 1.1 价值主张

**核心价值：**
```
"不是给你的大脑扩容，而是给你的思维建索引"

- 构建个人知识网络，让知识从碎片到体系
- AI 学习伴侣，让学习从孤独到陪伴
- 创作素材库，让创作从挤牙膏到如有源之水
```

**差异化价值：**
| 竞争对手 | 我们 | 差异化 |
|----------|------|--------|
| Notion | 知识图谱 + AI | 可视化关联、主动推荐 |
| Zotero | AI 伴侣 | 对话学习、成长记忆 |
| Obsidian | 多端 + AI | 移动友好、原生 AI |
| ChatGPT | 个人知识库 | 记忆持久、专业领域 |

### 1.2 客户细分

| 渠道 | 策略 | 成本 | 转化率预估 |

## 三、竞争优势与护城河

### 3.1 竞争优势

**短期优势（产品）：**
```
1. 知识图谱可视化 - 差异化核心
2. AI 伴侣记忆 - 竞争壁垒
3. 中文优化 - 本地化优势
```

**中期优势（数据）：**
```
1. 用户学习数据积累 - 网络效应
2. 知识图谱规模 - 越用越准
3. 领域深度 - 学术/技术专业版
```

**长期优势（生态）：**
```
1. Agent 市场 - 平台化
2. 品牌认知 - 品类代名词
3. 专利与技术 - 核心算法
```

### 3.2 护城河模型

```
┌─────────────────────────────────────────────────────────┐
│                      第四层：生态护城河                   │

│   用户越多 → 数据越多 → AI 越准 → 价值越高               │
```

## 四、实施路径

### 4.1 阶段规划

#### 阶段一：MVP 验证（0-6 个月）

**目标：** 验证产品价值，找到 PMF

```
里程碑：
Month 1: 技术 POC 完成
Month 2: Alpha 版本上线
Month 3: Beta 版本内测（1000 用户）
Month 6: 正式版发布（1 万用户）

关键指标：
- 用户留存率：40%（周留存）
- 付费转化率：5%
- NPS 分数：40+
- 文档上传量：人均 20+
```

#### 阶段二：快速增长（6-18 个月）

**目标：** 扩大用户规模，建立品牌

```
里程碑：
Month 9: 用户突破 10 万
Month 12: 完成 A 轮融资
Month 18: 用户突破 50 万

关键动作：
- 加大内容营销（小红书、B 站）
- 推出 Team 版本
- 建立学术/技术垂直社区
- 启动 API 生态
```

#### 阶段三：商业化深化（18-36 个月）

**目标：** 多元化收入，实现盈利

```
里程碑：
Month 24: 企业版发布
Month 30: Agent 市场上线
Month 36: 收入突破 2000 万/月

关键动作：
- 企业销售团队
- Agent 开发者生态
- 国际化准备
- 考虑收购/合并
```

### 4.2 团队规划

| 阶段 | 团队规模 | 核心角色 |
|------|----------|----------|
| MVP | 3-5 人 | 产品、技术 |
| 成长期 | 15-20 人 | + 设计、运营、营销 |
| 成熟期 | 50-100 人 | + 销售、企业、HR |

**早期团队分工：**
```
CEO/产品
├── CTO（技术负责人）
│   ├── 前端 1-2 人
│   ├── 后端 1-2 人
│   └── AI/算法 1 人
└── COO（运营）
    ├── 内容运营 1 人
    └── 用户运营 1 人
```
