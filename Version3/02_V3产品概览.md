## 产品一句话定位

**灵犀智读** — 首款基于大模型 Agent 的个人终身学习数字伴侣

---

## 核心差异

```
不是给你的大脑扩容，而是给你的思维建索引
不是帮你读更多书，而是让你读过的书都记住
不是替代你思考，而是放大你的思考能力
```
